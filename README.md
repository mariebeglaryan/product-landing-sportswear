# product-landing-sportswear

A responsive sportswear showcase landing page (HTML/CSS/JS) built to practice a real GitHub workflow.

## Run locally
Open `index.html` in a browser (or use VS Code Live Server).

## Structure
- `index.html` — page layout
- `styles.css` — styling + responsive layout
- `app.js` — basic JS behaviors
- `assets/` — images/icons (optional)
